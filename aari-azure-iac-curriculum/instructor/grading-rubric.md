# Grading Rubric

## Suggested Categories

### 1. Environment Setup

- student can explain the purpose of Azure CLI and Pulumi
- student can activate the Python environment correctly
- student can run `pulumi preview`

### 2. Infrastructure Understanding

- student can explain what the lab builds
- student can explain why the resources matter
- student can identify tags in the code

### 3. Code Quality

- no hardcoded secrets
- region is `eastus`
- code remains readable
- resources are named clearly

### 4. Operational Discipline

- student includes cleanup steps
- student explains cost concerns
- student understands preview before deploy

### 5. Reflection

- student connects the lab to AI, DevOps, or robotics infrastructure
- student can describe what they observed in the preview output

## Suggested Scoring

- 25% setup and commands
- 25% infrastructure understanding
- 20% code quality
- 15% cleanup and cost discipline
- 15% reflection
